A Complete AI Framework For Unity
Easy To Customize

Total AI's comprehensive ScriptableObject Type system allows for quick and easy customization. Simply inherit from the Type to be modified, implement the key methods, and then plug in your new Type.

Customizing Video
Powerful And Flexible AI

Total AI builds a strong foundation for Agent-based AI. Drives and their utility make the top layer and determine which group of Plans an Agent should consider. GOAP, Utility AI, and FSM+ are the currently implemented Planners. Target Factors, which use principles of Utility AI, can dynamically select targets.

Mappings & Plans

Total AI's core unit of a Plan is a Mapping. It has InputConditions that must be true for it to be performed and OutputChanges which occur at various timings during the running of the Mapping. Plans combine Mappings into a tree structure similar to GOAP.

Planning Video
Intelligent Customizable Agents

Agents are divided into logical parts: Planner, Decider, Sensors, Memory, Movement, and Animation. Each part is easy to customize and an be different for each Agent. Agents abilities and motivations are defined by their Actions, Drives, and Attributes. They also have a general inventory system which uses a Utility AI like system to pick inventory items.

Agent Overview Video
World Objects

World Objects are non-Agent object that exists in the world. They can be built, can grow, can be damaged, can have states, and can change appearance based on any of these factors. They also have inventory and can convert inventory using recipes.
