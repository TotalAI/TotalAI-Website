ABOUT

Mission Statement

Total AI is an MIT licensed open source agent AI Framework for Unity. Its goals are to provide an easy to prototype, flexible, fully customizable, and performant framework for a broad array of AI types and for a broad array of game types.

You Can Contribute!

Come join the Total AI Development Community! Submit a bug report or a feature request, build a new feature, fix a bug, or help with documentation. All contributions are welcome.
