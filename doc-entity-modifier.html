Entity Modifier

What is an Entity Modifier?

A Entity Modifier is a Scriptable Object attached to an Entity. It allows Output Changes to be triggered when certain events occur and/or Input Conditions are met.

Trigger Options

There are currently 10 trigger options:

  • Main Loop:
  • Update Loop:
  • Level Change:
  • On Trigger Enter:
  • On Trigger Exit:
  • On Trigger Stay:
  • On Collision Enter:
  • On Collision Exit:
  • On Collision Stay:
  • On Particle Collision:

For Target

Flips the Entity and the target Entity causing the target to run the ICs and OCs. Helpful if Entity is a World Object that impacts a target Agent since many ICs and OCs are created for Agents to run them.

Unavailable Output Change Features

Not all of the features available to an Output Change attached to a Mapping Type are available to an Output Change attached to an Entity Modifier.
