InputCondition

What is an InputCondition?

An Input Condition is a plain serializable C# class. It is used by Mapping Types, Entity Modifiers, Agent Type's Animation Overrides, and Role Types. It holds the values used by its Input Condition Type (ICT). The ICT defines what values are required and thus how to draw it in the Editor.

See Input Condition Type.
